üéÆ How to Use

  1. Add Custom Content Type:
    • Go to Dashboard ‚Üí Add Channel or edit existing channel
    • In the Content Type section, click the "+" button
    • Enter your custom content type name (e.g., "Tech Reviews", "Cooking Tutorials")
    • Click "Save" to add it to your available types
  2. Use Custom Content Type:
    • Select your custom type from the dropdown
    • It will appear with "(Custom)" label to distinguish from predefined types
  3. Manage Custom Types:
    • View all your custom types in the "Custom Content Types" section
    • Click the "√ó" button next to any custom type to delete it
    • If you delete a type that's currently selected, it will switch to "Other"

üíæ Data Persistence

Note: Custom content types are stored in your browser's localStorage under user settings. They will persist across browser sessions and be included in data exports/backups.
